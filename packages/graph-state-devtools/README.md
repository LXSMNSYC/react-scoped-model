# graph-state-devtools

## Install

- Download [latest release ZIP](https://github.com/LXSMNSYC/scoped-model/releases/tag/graph-state-devtools%400.1.2)
- Unpack ZIP
- Go to `chrome://extensions/` > `Load Unpacked`
- Select folder of the unpacked ZIP
